    2.0 (2011/04/14) - Majority of the extension’s code has been rewritten making it far better than previous versions. Every aspect of Recent History has been improved in some way. Please read the list below for all changes made to Recent History.

    Pop-up:
    - Tweaked layout and added a bit of colour.
    - You can now change the theme of the pop-up.
    - The search has been upgraded to instant search.
    - Time has been removed as it is irrelevant.
    - Items can be pinned, deleted and shared.
    - You can change the style of the items.
    - More icons have been added.
    - You can now filter domains you do not want to see in the recent history list.
    - The left click function can now be changed.

    Options Page:
    - The options page has been overhauled with a new design and UI.
    - Added a twitter page so you can keep up to date with what’s going on.
    - Added a translations page so you can easily send new or better translations directly.

    History Page:
    - The history page now also has instant search.
    - You can choose what you would like to search.
    - A Google web history styled calendar has been added for easier navigation.
    - You can now shift select items for deleting.
    - Items can now be grouped according to domain names.

    1.9.6 (2011/02/04) - Fixed issue of the middle-click function not working in the new stable release of chrome (9.0.597.84).

    1.9.4 (2011/01/17) - Fixed issue of the most visited list not being displayed. If this release is an upgrade for you then please restart Chrome for the fix to be applied.

    1.9.3 (2010/12/29) - Fixed a few bugs and cleaned up the code a little.

    1.9.2 (2010/12/22) - You can choose what icon you would like in the address bar for Recent History in the options page. Items which have been bookmarked will show a star next them in the history page. The middle-click function has been changed back so when you middle-click an item in the popup window the item opens in a new tab in the background and leaves the popup open.

    1.9.1 (2010/12/20) - Fixed a few bugs with the most visited list. The new history page is the default history manager but this can be changed in options. The new history page shows more search items from the past and not just from your recent history. A few things from the options page have been changed.

    1.9 (2010/12/16) - A brand new history page has been made which can be accessed by clicking on the "Show All History" link. Polish translation has been added, thanks to the person who commented. When you left click an item in the popup menu it opens in the current tab and if you middle click the item, it will open up in a new tab. When the popup or the history page is opened the search bar is focused so you search without clicking on the search field. The recently closed tabs history isn't deleted upon new browser session any more. When you hover over a recent history or most visited item you will be able to see the visit count for that URL. The most visited history items are now gathered from the past 4 weeks which should give you an accurate list of most visited URLs. You can also remove the URLs you do not want to show up in the most visited list in the options page. The popup should open up slightly quicker after you open it the first time. Lastly a few minor changes have been made here and there.

    1.8.2 (2010/06/11) - Fixed issue of the extension causing a high cpu load on some systems like Mac OS and Linux. Please make sure your using the latest stable version of Chrome.

    1.8.1 (2010/06/06) - Fixed issue of time being displayed even if the option is switched off.

    1.8 (2010/06/06) - Added new history API so the recent history you see is from chrome's actual history. This extension still needs to track your history for recently closed tabs however. I've added a recent bookmarks section also. Some options had to be removed due to either it not being allowed, better efficient coding and mainly due to the addition of chrome 5 APIs. Languages are still supported, however the only way you can use them is to install the locale version of chrome. This is the official way to support localization in chrome extensions. The history manager has been removed for now however I'm working on another new one which will act as your default history page.

    1.7.4 (2010/05/25) - Fixed issue of no favicons and titles showing up.

    1.7.1 (2010/04/27) - Fixed clear extension cache button on the popup menu.

    1.7 (2010/04/25) - Added most visited links, option to order the sections for the popup menu, option to remove the search box and a button on the menu to clear extension cache. You can now also input your own values in the options page. Fixed the search clear button being hidden by the scroll bar and fixed issue of repeated history items in the recently closed tabs section. Lastly the extension is now able to record your browsing history item before the page is fully loaded.

    1.6 (2010/03/09) - Added a search feature in the popup menu and you can now view a maximum of 100 history items in the history manager. Also fixed the issue of popup menu not showing up due to favicon url being in a restricted area (i.e. you have to be logged in).
